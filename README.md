# pili

## Architecture

```py
app/
    front/
        dist/        # front buildé
        src/         # fichiers source du front
        index.html   # point d'entrée principal du front
    back/
        src/         # fichiers source du back
        index.js     # point d'entrée principal du back
```

## Techs

- Front : React buildé avec Vite
- Back : Node + Express
- Lang : JS
- API : Websockets & JSON
